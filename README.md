# Tower-Filter-MOD
为《极简塔防（Minimalist Tower Defense）》制作的 UI 增强模组，在选塔界面加入【分类 + 等级】筛选功能，提升后期与高难度局的操作效率。基于 BepInEx 与 Harmony，不修改游戏原文件。A UI enhancement mod for Minimalist Tower Defense that adds category and rarity filters to the tower selection panel, improving usability in late-game and high-difficulty runs. Built with BepInEx and Harmony.
